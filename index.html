Mongoid

Mongoid provides an elegant way to persist and query Ruby objects to documents in MongoDB.

Mongoid 1.x for
Rails 2

gem install mongoid

Mongoid 2.x for
Rails 3

gem install mongoid --pre
Ok, Impress Me »

Features

  • Effortless mapping of Ruby objects to documents and embedded documents.
  • Rails 3 compatibility including support for ActiveModel and Generators.
  • Rich criteria API for document retrieval from the database.
  • Named Scopes / Chainable criteria.
  • Versioning of your data/documents.
  • Full callback support on documents and embedded documents.
  • Proper master/slave distribution against the database.
  • Optimized for use with extremely large datasets.

Rich Criteria API

.all(:aliases => ["The Dude", ...])

.and(:age.gt => 18)

.excludes(:status => "Married")

.in(:status => ["Single", "Divorced"])

.where(:age.lte => 55)

Chainable Criteria & Named Scopes

Person
  .where(:last_name => "Zorg")
  .and(:middle_initial => "J")
  .and(:age.gt => 30)
class Player
  include Mongoid::Document
  field :status
  scope :inactive,
        :where => { :active => false }
end

Master/Slave support

# all reads go to slaves
class Person
  include Mongoid::Document
  enslave
end
# force a single query to use slaves
.where(:name => "Durran").enslave
