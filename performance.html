performance

Performance numbers are recorded for each release as of 2.0.0 and displayed here. The data used in the metrics is that of what would be considered a fairly standard application in MongoDB that Mongoid could manage.

  • 1,000,000 documents in the main collection
  • 1,000 embedded documents on a root document (1-n)
  • 100,000 referenced documents to another collection (1-n)
  • 10,000 referenced documents to another collection (n-n)

* Performance metrics done on a 2.93 GHz i7 iMac / 8GB RAM.

  • Green indicates fast operations
  • Yellow indicates average operations
  • Red indicates slow operations
